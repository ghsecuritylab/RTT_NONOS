# RTT_NONOS

RTT_NONOS基于RT-Thread 4.0.1，是RT-Thread的裸机版本。使用 **状态机编程** 。

目前RT_NONOS支持的组件有 **shell** 、 **AT Client** 、 **Ymode** 、 **SFUD** 等组件。

目前支持的软件包测试过的有 **FAL** 、 **EASYFLASH**。

[Project](./Project) 文件夹下的是目前所实现的工程。
